# M-DOS  OPENAPI  Documentation

Welcome to the M-DOS OPENAPI documentation. M-DOS (Medical-Dynamic Operating System) provides a set of APIs to interact with its platform. These APIs enable developers to manage tasks, projects, and workflows activities within MDOS efficiently.

## Version

Current Version: **1.0.0**

## Authentication

Authentication to the M-DOS API is performed via API key (Token) authentication. Include your API key in the `Authorization` header of your requests.

## Date And Time Format
All time parameters in the M-DOS system are formatted according to the ISO 8601 international standard.

## Base URL

All endpoints are relative to the following base URL: `https://local.mdos.intrii.com/openapi`
